Claude AI skill for generating modern, non-generic, high-quality websites with clean UI/UX and production-ready code.

🛠️ Usage

Inside Claude Code, activate the skill using:

/no-generic-web

💡 Recommended (token-efficient usage)

For better performance and lower token consumption, use:

/caveman /no-generic-web Create a landing page for a digital agency

👉 Caveman Mode is strongly recommended for:

* faster responses
* reduced token usage
* more efficient iterations

⸻

Example prompts

/caveman /no-generic-web Luxury Mode Create a high-end architecture studio website
/caveman /no-generic-web Luxury Mode and 3D Scroll Mode Create a premium interactive website

## 📦 Installation

### Option 1 — Download the `.skill` file

1. Download `no-generic-web.skill` from [Releases](https://github.com/santiagojuncal/-claude-no-generic-web/releases)
2. Extract it into your Claude skills folder:

**Windows:**
```cmd
mkdir %USERPROFILE%\.claude\skills\no-generic-web
tar -xf no-generic-web.skill -C %USERPROFILE%\.claude\skills\no-generic-web
```

**Mac / Linux:**
```bash
mkdir -p ~/.claude/skills/no-generic-web
unzip no-generic-web.skill -d ~/.claude/skills/no-generic-web
```

### Option 2 — Clone with Git

**Windows:**
```cmd
git clone https://github.com/santiagojuncal/-claude-no-generic-web %USERPROFILE%\.claude\skills\no-generic-web
```

**Mac / Linux:**
```bash
git clone https://github.com/santiagojuncal/-claude-no-generic-web ~/.claude/skills/no-generic-web
```

3. Restart Claude Code — the skill loads automatically.
